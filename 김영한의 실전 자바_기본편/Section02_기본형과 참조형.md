# Section02\_기본형과 참조형

## 변수와 초기화

- 변수의 종류
  - 멤버 변수(필드): 클래스에 선언
  - 지역 변수: 메서드에 선언, 매개변수도 지역 변수의 한 종류
- 변수의 값 초기화
  - 멤버 변수는 자동 초기화
    - 인스턴스의 멤버 변수는 인스턴스를 생성할 때 자동으로 초기화
    - 숫자 = 0 / boolean = false / 참조형 = null
  - 지역 변수는 직접 초기화 해야 한다.

## NullPointerException

- Null 값에 .(dot)을 찍었다고 생각하면 쉽게 디버깅 할 수 있다.

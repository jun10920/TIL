# Section09\_메서드

## TIP

- ctrl + b

  - 해당 메서드가 정의된 곳으로 이동

- 인수(Argument)
  - 함수를 호출할 때 넘기는 값
- 매개변수(Parameter)
  - 메서드를 정의할 때 선언되는 매개변수
- 반환형이 void인 경우
  - return문을 자동으로 삽입하기 때문에 생략 가능
- **자바는 항상 변수의 값을 복사해서 대입한다.**
- 메서드 오버로딩
  - 메서드는 이름이 같고 매개변수가 다른 오버로딩이 가능하다.
  - 매개변수의 타입과 순서가 다르면 오버로딩인데, 반환 타입은 인정하지 않는다.
- 메서드 시그니처
  - 메서드 이름 + 매개변수 타입(순서)
  - 메서드를 구분하는 기준
